# kbsecret-website

[![Build Status](https://travis-ci.org/kbsecret/kbsecret-website.svg?branch=master)](https://travis-ci.org/kbsecret/kbsecret-website)

Depends on node & npm

to install:

```
git clone
cd into it
npm install
```

to build:

```
npm run build
```

to run locally: (after building!)

```
npm start
```
